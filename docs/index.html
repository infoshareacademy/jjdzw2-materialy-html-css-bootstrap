Inne konstrukcje 


                    class Rectangle {
                        constructor(height, width) {
                            this.height = height;
                            this.width = width;
                        }
                        
                        get area() {
                            return this.height * this.width;
                        }
                    }

                    for (let i = 0; i < 5; ++i) {
                        const a = Math.floor((Math.random() * 10) + 1);
                        const b = Math.floor((Math.random() * 10) + 1);
                        const rect = new Rectangle(a, b);

                        alert(`${a} x ${b} = ${rect.area}`);
                    }

DOM – drzewo

Eventy

  • Powiadamianie użytkownika o jakimś zajściu
  • Eventy myszy: click, dblclick, mouseover
  • Eventy klawiatury: keydown, keyup
  • Eventy formatek: blur, focus, change
  • Eventy schowka: copy, paste
  • Duużo innych + informacje o poszczególnych eventach: w3schools

Nasłuchiwanie na eventy

  • HTML style:
    • <button id="button" onclick="funkcja()">
  • Vanilla JS:
    • document.getElementById('button') .addEventListener('click', funkcja);
    • document.getElementById('button') .onclick = funkcja;
